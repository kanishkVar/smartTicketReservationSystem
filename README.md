Smart Ticket Reservation System.
Tech: Java (Swing/JavaFX), JDBC, MySQL (or SQLite).
How to run:
1. Import project in IDE.
2. Create DB: run `schema.sql`.
3. Update DB credentials in `src/utils/DatabaseConnection.java`.
4. Run `Main.java`.

Features:
- Add / View / Delete trips
- Book tickets (auto seat allocation)
- Simple GUI + JDBC
